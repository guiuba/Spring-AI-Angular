# Spring AI + Angular Project

This project is a development based on the playlist **"Spring AI + Angular"**, kindly offered by **Loiane Groner** (thank you so much Loiane).  
It is a sample application that showcases a **Java Spring AI backend** integrated with an **Angular frontend**.  
The project demonstrates the integration of **Spring AI services (chats)** within an Angular interface.
